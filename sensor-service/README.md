# Sensor Service

Offers basic CRUD functionalities for sensors and measurements.
Furthermore, mean values of sensor measurements can be retrieved via the appropriate GET-endpoint.
Uses a MongoDB database and Spring Data Mongodb for a simple implementation.

## Starting the Application

1. Navigate to **src/main/java/com.microservice.sensor**
2. Open **SensorServiceApplication** class
3. **Run** SensorServiceApplication

## Swagger/API Documentation

In order to have a look at the provided API -> open browser with URL **http://localhost:port/swagger-ui/index.html**
