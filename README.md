# Starting the Application

The following steps need to be done in the specified order so that the application runs as intended.

1. **Docker Desktop**
    - Start Docker Desktop on your machine
2. **Terminal**
    - Open a terminal in your preferred IDE
    - Run command _**"docker compose up -d"**_
    - Wait for Images to pull from docker hub
    - Once every container has been _created_ **AND** _started_ --> startup process done
      - check via Docker Desktop --> status of all containers should be "**Running**"
