# Hotel Service

Offers basic CRUD functionalities for hotels.
Uses a MongoDB database and Spring Data Mongodb for a simple implementation.

## Starting the Application

1. Navigate to **src/main/java/com.microservice.hotel**
2. Open **HotelServiceApplication** class
3. **Run** HotelServiceApplication

## Swagger/API Documentation

In order to have a look at the provided API -> open the browser with URL **http://localhost:port/swagger-ui/index.html**
