# Config Server

Makes use of Spring Config Server which is designed to centralize configuration
management for microservices-based applications. It provides a centralized and
version-controlled configuration source, allowing the management of application
configurations across multiple environments efficiently.

## Starting the Application

1. Navigate to **src/main/java/com.microservice.config**
2. Open **ConfigServerApplication** class
3. **Run** ConfigServerApplication

## Add new configurations

1. Visit
   **https://github.com/hochschule-jz/gruppe-9-microservice-config-server**
2. Add new **.properties** file on main branch -> filename should equal
   **spring.application.name** of service
